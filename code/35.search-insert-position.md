# [Search Insert Position](https://leetcode.com/problems/search-insert-position/description/)

首先最简单的，直接暴力查找即可。

```go
func searchInsert(nums []int, target int) int {
	for i, n := range nums {
		if n >= target {
			return i
		}
	}
	return len(nums)
}
```

但是这样显然不符合题目对于时间复杂度 `O(log n)` 的要求。

题目中给定的数组是已经排序后的，所以自然地，我们可以考虑二分法。

```go
func searchInsert(nums []int, target int) int {
	li := 0             // left index
	ri := len(nums) - 1 // right index
	for li != ri {
		mid := (ri-li)/2 + li
		if li+1 == ri {
			if target == nums[li] {
				return li
			}
			if target == nums[ri] {
				return ri
			}
			if target > nums[ri] {
				return ri + 1
			}
			if target < nums[li] {
				return ri - 1
			}
			return ri
		}
		if nums[mid] == target {
			return mid
		}
		if nums[mid] < target {
			li = mid
		} else {
			ri = mid
		}
	}
	if nums[li] >= target {
		return li
	}
	return li + 1
}
```
