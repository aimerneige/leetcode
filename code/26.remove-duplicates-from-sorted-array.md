# [Remove Duplicates from Sorted Array](https://leetcode.com/problems/remove-duplicates-from-sorted-array/description/)

本题非常简单，循环检查，当出现不一样的数字就移动赋值到数组前面即可。

```go
func removeDuplicates(nums []int) int {
	index := 0
	for _, num := range nums {
		if num != nums[index] {
			index++
			nums[index] = num
		}
	}
	return index + 1
}
```
